ん音，两个nn就是ん

拗音，第一个假名的第一个音节+第二个假名的两个音节

促音，促音后面的假名的辅音打两次,例如：kku，っく

长音，そう/そ——（shift+"——"）


Alt  + CapsLock是平假名输入

Ctrl + CapsLock是片假名输入
